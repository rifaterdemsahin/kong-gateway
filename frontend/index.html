Welcome to the Smart-Cost demo — running entirely in your browser.

The same keyword classifier Kong uses in its Lua plugin is replicated here in JavaScript. Send a message and watch it get routed to the right model automatically:

● GPT-4o-mini — lightweight tasks (Q&A, summaries, translation)
● Claude 3.5 Sonnet — complex tasks (code, debugging, analysis)

Every routing decision shows you the cost saved vs. using Claude for everything.
